###########################
#Windows
###########################
Move the runme_windows.bat file to the src folder, then double click on the 
runme file to run. The script will compile the program, then open a console 
where you can communicate with the chatbot.

###########################
#MacOs/Linux
###########################
Move the runme_macos-linux.sh file to the src folder, open a terminal in the
src folder, then type

sh runme_macos_linux.sh

to compile and run the program. You can communicate with the chatbot directly
in the terminal.
